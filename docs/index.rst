HDF5 Python 中文文档
===============

h5py包 是HDF5二进制数据格式的Pythonic接口。

`HDF5 <https://hdfgroup.org>`_ 可以存储大量的数字数据，并轻松地从NumPy中操作这些数据。例如，您可以将存储在磁盘上几TB的数据集切片，就像它们是真正的NumPyarrays一样。数千个数据集可以存储在一个文件中，根据需要进行分类和标记。
